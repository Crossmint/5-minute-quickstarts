## Quick Setup

```bash
git clone https://github.com/Crossmint/5-minute-quickstarts && cd 5-minute-quickstarts && ./setup.sh wallets-react
```

This will install dependencies, authenticate with Crossmint, and start the dev server.

## What This Demonstrates

- **Authentication** — Email-based login/logout
- **Wallet creation** — Auto-creates a wallet on first login
- **Fund wallet** — Add testnet USDXM tokens
- **Check balance** — View wallet balance
- **Transfer funds** — Send tokens to another address

## Go to Production

1. Create a project at [crossmint.com/console](https://www.crossmint.com/console)
2. Generate a client API key with wallet scopes
3. Replace your staging key and switch to a mainnet chain
